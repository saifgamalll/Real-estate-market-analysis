# Residential Real Estate Market Analysis

## 🎯 Project Overview
This project analyzes the factors that influence housing prices, focusing on identifying the core drivers of market value. It translates raw property data into a predictive model for real estate valuation.

## 🛠️ Analytical Approach
* **Feature Engineering:** Identified high-impact variables such as location density, square footage, and historical price trends.
* **Data Visualization:** Created distribution plots and heatmaps to visualize the correlation between property features and final sale prices.
* **Regression Modeling:** Utilized Linear Regression and Random Forest techniques to predict market outcomes.
* **Outlier Detection:** Filtered out skewed data points (extreme luxury properties) to ensure the model reflects the "average" consumer market.

## 📈 Impact
The model provides a clear understanding of which features (e.g., renovation status vs. location) contribute most to price appreciation.
